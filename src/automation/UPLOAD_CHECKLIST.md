# YouTube 手動投稿チェックリスト

このチェックリストは、GitHub Actions で生成された動画を YouTube に手動投稿する際の確認事項です。

## 📥 1. Artifacts のダウンロード

### 1.1 GitHub Actions 実行履歴から取得

```bash
# Web UI から
https://github.com/apol0510/keiba-ai-predictor/actions
→ 最新の "Daily Video Generation Automation" をクリック
→ "Artifacts" セクションから "daily-videos-X" をダウンロード
```

### 1.2 ダウンロードしたファイルを解凍

```
daily-videos-X.zip
├── shorts_YYYYMMDD.mp4           # Shorts 動画
├── youtube_YYYYMMDD.mp4          # 通常動画
├── full_version_YYYYMMDD.mp4     # 長編動画
├── thumbnail_shorts_YYYYMMDD.png # Shorts サムネイル
├── thumbnail_youtube_YYYYMMDD.png# 通常動画サムネイル
├── thumbnail_full_YYYYMMDD.png   # 長編動画サムネイル
├── metadata_YYYYMMDD.json        # メタデータ（タイトル・概要欄・タグ）
├── prediction_data.json          # 予想データ（参考用）
└── generation_summary.json       # 生成サマリー（参考用）
```

## 📱 2. Shorts 投稿（1本目）

### 2.1 投稿前確認

- [ ] **動画の確認**
  - [ ] 動画ファイル: `shorts_YYYYMMDD.mp4`
  - [ ] サイズ: 約 200-300KB
  - [ ] 再生時間: 30秒
  - [ ] アスペクト比: 9:16 (縦型)
  - [ ] 音声: ナレーション + BGM が聞こえる
  - [ ] 画面: 競馬場名・レース番号・本命馬が表示される

- [ ] **サムネイルの確認**
  - [ ] サムネイルファイル: `thumbnail_shorts_YYYYMMDD.png`
  - [ ] 色: 赤系（Shorts 用）
  - [ ] テキスト: レース情報と本命馬が読める

- [ ] **メタデータの確認**
  - [ ] `metadata_YYYYMMDD.json` を開く
  - [ ] `.shorts.title` を確認
  - [ ] `.shorts.description` を確認
  - [ ] `.shorts.tags` を確認

### 2.2 YouTube Studio で投稿

1. **YouTube Studio を開く**
   - https://studio.youtube.com

2. **動画をアップロード**
   - "作成" → "動画をアップロード"
   - `shorts_YYYYMMDD.mp4` を選択

3. **メタデータ入力**
   - [ ] **タイトル**: `metadata.shorts.title` をコピペ
     ```
     例: 【衝撃】◎4番ジーティーラピッドが激走！川崎11R AI予想
     ```

   - [ ] **説明**: `metadata.shorts.description` をコピペ
     ```
     例: 🏇 2026年3月6日 川崎競馬 11R AI予想

     ◎本命: 4番 ジーティーラピッド (35.0%)
     ...
     ```

   - [ ] **サムネイル**: `thumbnail_shorts_YYYYMMDD.png` をアップロード

4. **視聴者設定**
   - [ ] 子ども向けではない: 選択
   - [ ] 年齢制限: なし

5. **その他の設定**
   - [ ] **タグ**: `metadata.shorts.tags` から手動で追加
     ```
     競馬予想, AI予想, 競馬AI, 川崎競馬, 本命, 的中, 競馬, shorts, 競馬shorts, 激走, 穴馬
     ```

   - [ ] **カテゴリ**: スポーツ

   - [ ] **言語**: 日本語

   - [ ] **Shorts として認識されているか確認**
     - 縦型動画（9:16）、60秒以下 → 自動的に Shorts になる

6. **公開設定**
   - [ ] **公開**: 公開
   - [ ] **公開予約**: 必要に応じて設定
   - [ ] **プレミア公開**: OFF

7. **投稿完了**
   - [ ] "公開" ボタンをクリック
   - [ ] 投稿URL をメモ

## 📺 3. 通常動画投稿（2本目）

### 3.1 投稿前確認

- [ ] **動画の確認**
  - [ ] 動画ファイル: `youtube_YYYYMMDD.mp4`
  - [ ] サイズ: 約 600-800KB
  - [ ] 再生時間: 70-90秒
  - [ ] アスペクト比: 16:9 (横型)
  - [ ] 音声: ナレーション + BGM が聞こえる
  - [ ] 画面: 複数レースが順次表示される

- [ ] **サムネイルの確認**
  - [ ] サムネイルファイル: `thumbnail_youtube_YYYYMMDD.png`
  - [ ] 色: 青系（YouTube 用）
  - [ ] テキスト: 本日の競馬場・日付・本命馬が読める

- [ ] **メタデータの確認**
  - [ ] `metadata_YYYYMMDD.json` を開く
  - [ ] `.youtube.title` を確認
  - [ ] `.youtube.description` を確認
  - [ ] `.youtube.tags` を確認

### 3.2 YouTube Studio で投稿

1. **動画をアップロード**
   - "作成" → "動画をアップロード"
   - `youtube_YYYYMMDD.mp4` を選択

2. **メタデータ入力**
   - [ ] **タイトル**: `metadata.youtube.title` をコピペ
     ```
     例: 【AI予想】川崎競馬 3/6｜本命◎4番ジーティーラピッドが狙い目！
     ```

   - [ ] **説明**: `metadata.youtube.description` をコピペ
     ```
     例: 🏇 2026年3月6日 川崎競馬 AI予想

     本日の注目レースTOP3をAIが徹底分析！
     ...
     ```

   - [ ] **サムネイル**: `thumbnail_youtube_YYYYMMDD.png` をアップロード

3. **視聴者設定**
   - [ ] 子ども向けではない: 選択
   - [ ] 年齢制限: なし

4. **その他の設定**
   - [ ] **タグ**: `metadata.youtube.tags` から手動で追加
     ```
     競馬予想, AI予想, 競馬AI, 川崎競馬, 本命, 的中, 競馬, 競馬予想AI, 今日の競馬, 競馬本命, AI競馬
     ```

   - [ ] **カテゴリ**: スポーツ

   - [ ] **言語**: 日本語

5. **動画の要素**
   - [ ] **終了画面**: 動画終了10秒前に追加
     - チャンネル登録ボタン
     - 関連動画（最新のアップロード動画）

   - [ ] **カード**: 追加（オプション）
     - Shorts 動画へのリンク
     - 詳細予想サイトへのリンク

6. **公開設定**
   - [ ] **公開**: 公開
   - [ ] **公開予約**: 必要に応じて設定（14:00 など）
   - [ ] **プレミア公開**: OFF

7. **投稿完了**
   - [ ] "公開" ボタンをクリック
   - [ ] 投稿URL をメモ

## 🔗 4. 投稿後の作業

### 4.1 Shorts と通常動画を相互リンク

- [ ] **Shorts の説明欄に通常動画のリンクを追加**
  ```
  📺 詳しい予想はこちら
  https://youtu.be/XXXXXXXXXX
  ```

- [ ] **通常動画の説明欄に Shorts のリンクを追加**
  ```
  📱 ショート動画もチェック！
  https://youtube.com/shorts/XXXXXXXXXX
  ```

### 4.2 コミュニティ投稿（オプション）

- [ ] YouTube Studio → "コンテンツ" → "投稿"
  ```
  🏇 本日の川崎競馬 AI予想を公開しました！

  ◎本命: 4番 ジーティーラピッド (35.0%)

  📺 動画で詳しく解説しています
  [動画リンク]
  ```

## ⚠️ 5. 投稿時の注意事項

### 5.1 YouTube ポリシー遵守

- [ ] **賭博関連コンテンツ**
  - ✅ 予想情報の提供: OK
  - ❌ 賭博の推奨・販売: NG
  - ❌ 未成年への訴求: NG

- [ ] **誤解を招く表現を避ける**
  - ❌ "100%的中"
  - ❌ "絶対勝てる"
  - ✅ "AI予想"、"的中率80%超え"（実績ベース）

### 5.2 投稿タイミング

- [ ] **推奨投稿時間**
  - レース当日 12:00 - 14:00（レース開始前）
  - 前日 20:00 - 22:00（翌日の予想）

- [ ] **避けるべき時間**
  - レース終了後（結果が出てから）
  - 深夜帯（視聴者が少ない）

### 5.3 品質確認

- [ ] **動画の最終確認**
  - [ ] 音声が明瞭に聞こえる
  - [ ] 画面が見やすい（モバイルでも確認）
  - [ ] テキストが読みやすい
  - [ ] BGM が大きすぎない

- [ ] **メタデータの最終確認**
  - [ ] タイトルに誤字脱字がない
  - [ ] 日付・競馬場名が正しい
  - [ ] 説明欄のリンクが有効

## 🎯 7. 次回以降の改善点

投稿を重ねるごとに、以下の点を確認・改善してください：

- [ ] **タイトル A/B テスト**
  - どのタイトルがクリックされやすいか
  - 絵文字の効果

- [ ] **サムネイルの改善**
  - どの色・デザインが目を引くか
  - テキストサイズ・フォント

- [ ] **投稿時間の最適化**
  - どの時間帯が視聴回数が多いか

- [ ] **コンテンツの長さ**
  - Shorts: 30秒が最適か、もっと短い方がいいか
  - 通常動画: 90秒が最適か
